# Bombi Din

Bombi Din is an open-source typeface inspired by functional industrial typography, designed for clarity, structure, and modern digital use. It works well across interfaces, branding systems, and editorial layouts where legibility and personality need to coexist.

## Styles
- Bombi Din Regular  
*(Additional weights and styles may be added in future releases.)*

## Usage
You are free to use Bombi Din for:

- Personal and commercial projects
- Websites, apps, and UI design
- Branding, signage, and editorial layouts
- Print and digital media

You may also modify and redistribute the font under the terms of the license.

## Installation
1. Download the font files from the `fonts/` directory.
2. Install the `.otf` files on your system.
3. Use the font in any design or development environment.
